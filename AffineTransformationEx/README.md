# Using opencv affine transformation function to crop (then rotate) image by provided bbox


### Cpp Version

#### Compile & Run
```bah
 cd Cpp && mkdir build && cd build && cmake .. && make -j && cd .. && bash run.sh
```

### Scala Version

```bash
 cd Scala
```
then run `sbt` and compile the project

```bash
 bash run.sh
```

### Python Version
```bah
 cd Python && bash run.sh
```
